## ✨ Features

- 📅 **Dynamic Weekly Timetable**: Set up your daily lectures, labs, and breaks seamlessly.
- ✅ **One-Tap Marking**: Log your daily attendance straight from your customized schedule interface. Supported statuses include Attended, Missed, Holiday, and Cancelled.
- 📊 **Deep Analytics**: Check subject-wise breakdowns, overall percentages, minimum attendance targets, and advanced lecture modifier metrics directly from the Statistics Dashboard.
- 🏷️ **Modifiers System**: Tag specific lectures as Labs, Tutorials, or custom variations and track their individual attendance rates independently.
- 💾 **Granular JSON Backups**: Completely offline-first! Export specific academic sessions as JSON files directly to your Downloads folder, and import them seamlessly to retain complete data portability.
- 🎨 **Material 3 UI**: Full support for system-based Light/Dark modes, boasting vibrant neon indicator coloring, dynamic onboarding, and a gorgeous Calendar matrix.

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (`>=3.5.3`)
- Android Studio or VS Code with the Flutter extension installed

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/AttendMark.git
   cd AttendMark
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   Connect your Android device or start an emulator, then execute:
   ```bash
   flutter run
   ```
